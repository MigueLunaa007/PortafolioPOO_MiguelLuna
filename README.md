Este portafolio principal recopila los proyectos académicos y prácticos desarrollados durante mi formación en en el segundo semestre de la carrera de Ingeniería Informática. El objetivo de este espacio es consolidar soluciones de software reales, demostrando el aprendizaje continuo, la aplicación de fundamentos de ingeniería y la evolución de mis habilidades técnicas en desarrollo de software.

# Tecnologías utilizadas
* **Lenguajes de Programación:** PHP, Python
* **Desarrollo Web (Frontend):** HTML5
* **Entorno de Servidor Local:** XAMPP (Apache, MySQL)
* **Entorno de Desarrollo (IDE):** Visual Studio Code

# Competencias desarrolladas
* **Desarrollo de Aplicaciones Web:** Integración de lógica de servidor con interfaces de usuario mediante la conexión de PHP y HTML.
* **Pensamiento Algorítmico y Automatización:** Creación de scripts eficientes y resolución de problemas lógicos utilizando Python.
* **Gestión de Servidores Locales y Bases de Datos:** Configuración, administración y pruebas de entornos web locales y bases de datos relacionales a través de XAMPP.
* **Fundamentos de Ingeniería Informática:** Aplicación de buenas prácticas en la escritura de código, estructuración de archivos y uso óptimo de herramientas de desarrollo.
